# Quick Start

Get your first `yourdb` database running in under 2 minutes.

## 1. Define Your Data Model

`yourdb` works with your existing Python classes. Just add the `@register_class` decorator.

```python
from yourdb.utils import register_class

@register_class
class User:
    __version__ = 1  # Important for schema evolution!

    def __init__(self, name, email):
        self.user_id = None
        self.name = name
        self.email = email

    def __repr__(self):
        return f"User(id={self.user_id}, name='{self.name}')"
